# 写作类型：论文（Paper）

- **目标**：符合 arXiv 预印本的标准结构（中文撰写，附英文 Title / Abstract）。
- **必含章节**：标题(中+英) / 作者机构 / Abstract(英)+中文摘要 / 关键词 / 引言 / 方法 / 实验与结果 / 讨论 / 结论 / 参考文献。
- **格式**：
  - 正文 Markdown；公式用 LaTeX 片段（`$...$` / `$$...$$`），算法用代码块或伪代码。
  - 参考文献用 BibTeX（`@article{...}`）或 APA；文中引用用 `[1]` 或 `\cite{}`。
  - 图表用占位标题（如「图 1：...」「表 1：...」），用户后续替换。
- **必遵规则**：
  - 客观、严谨；克制第一人称，可用「本文 / 我们」。
  - 所有数据、指标、引述须可核查，**禁止编造**实验结果与数字。
  - 贡献点（contributions）在引言中明确列出。
  - 区分「本文工作」与「已有工作」，避免侵权式表述。
- **可选**：附录放补充实验；冲突利益声明。

> 这是写作类型的硬约束（harness），每次写论文都会加载。改这里即可调整全局论文规范。
