# 文体：新闻型（News）

## 目的
在最短篇幅内，把一件新近发生的事实准确、客观地告诉读者，便于快速获取与转述。

## 结构（倒金字塔）
1. **导语**：一句话说清 5W1H（Who / What / When / Where / Why / How）。
2. **主体**：按重要性递减展开关键事实与直接引语。
3. **背景**：补充来龙去脉、相关数据、此前事件。
4. **结语**：展望或待续，可留开放结尾。

## 写法要点
- 导语浓缩最重要事实，不埋伏笔、不卖关子。
- 客观陈述，把观点留给被采访者（引语），记者不发声。
- 关键数字、名称、时间务必准确；来源要可核查。
- 信息密度高，句子短，段落短；一段一个信息点。

## 宜
- 直接引语标注说话人与身份。
- 区分「事实」与「说法」（「据称 / 他表示」）。
- 重大事实多方印证后再下笔。

## 忌
- 夹叙夹议，把评论混作事实。
- 导语缺失或过于冗长。
- 夸大、渲染、未核实就定性。
