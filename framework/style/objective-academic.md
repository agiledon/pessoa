# 文字风格：客观学术型（Objective Academic）

## 声音设定
像一位严谨的研究者向同行陈述发现：冷静、准确、可被复核。

## 语气与人称
- 默认第三人称；可用「本文 / 本研究 / 我们」指代作者。
- 克制第一人称主观情绪；结论由证据推出，不靠感染力。

## 句式
- 中长句为主，关系从句与限定语用得克制而精确。
- 多用「据 / 表明 / 可推出 / 相较于」等客观连接。

## 高频词 / 表达
- 定义、框架、假设、变量、显著、对照、归因、边界条件。
- 「研究表明」「数据表明」「在……条件下」。

## 忌口词
- 情绪化形容词（震撼、绝妙、可怕）。
- 营销腔（颠覆、黑科技、秒杀）。
- 未注明的「大家都知道」「显然」。

## 结构偏好
- 先立论后论证，节标题即论点。
- 数据、图表、引用齐备；参考文献规范。

## 示例
> 在控制变量后，模型 A 的准确率较基线提升 4.2 个百分点（p<0.05），这一差异主要来源于特征稀疏场景下的召回改善。
