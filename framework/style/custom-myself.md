# 个人定制 · 我的写作声音档案（custom-myself）

> 这是你的通用写作声音模板。填写得越具体，Agent 模仿得越像你。
> 建议用你最代表自己的 3–5 篇真实文章提炼后填写；可随时增删。

## 1. 语气与人称
- 常用人称：第一人称「我」 / 第二人称「你」 / 第三人称 / 混合
- 态度基调：（如：平和、犀利、幽默、克制、热忱）
- 是否接受情绪与立场：是 / 否

## 2. 句式习惯
- 句长偏好：（如：多短句、长短交错、偶用长句）
- 标志性开头：（如：以提问起、以场景起、直接给结论）
- 连接习惯：（如：爱用「其实」「说白了」；少用「综上所述」）

## 3. 高频词 / 口头禅
- 常用：（如：说实话、踩坑、本质上、说白了）
- 偏爱比喻类型：（如：生活化类比、技术类比、历史典故）

## 4. 忌口词（绝不用 / 尽量少用）
- （如：颠覆、黑科技、赋能、值得注意的是、综上所述、令人震撼）

## 5. 结构偏好
- 开篇方式：（如：故事钩子、反差、直接点题）
- 收尾方式：（如：留白、金句、行动号召）
- 段落长度：（如：手机屏 3–5 行）
- 小标题密度：（如：300–500 字一个）

## 6. 示例段落（贴一段你写的、最像你的文字）
> （在此粘贴 1–2 段你的真实文字，作为 Agent 模仿的样例）
